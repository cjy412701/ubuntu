# chromium黑色模式

接下來呢~只要在網址列輸入指令：「chrome://flags/#enable-force-dark」，就會看到許多設定選項，找到「Force Dark Mode for Web Contents」，把「黑色模式」的設定打開，將「Disabled」 改為「 Enabled」，chrome就可以變更為黑色的背景模式了！
